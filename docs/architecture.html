热更流水线

.version -> .manifest -> hash/size 校验 -> 下载缺失 Bundle -> 注入 provider -> 加载 Asset -> 引用计数 -> Asset GC -> 废弃 Bundle 清理

打包模型

AssetBuildProfile233 配置多个 AssetGroup。每个 Group 下有多个 Collector。Collector 决定地址规则、标签、PackRule、压缩模式。构建结束后生成 AB、Manifest、Version、构建报告。

平台宏

微信小游戏只要求 WX。编辑器中没有 WX SDK 时仍能编译和模拟；所有微信 SDK 类型、using、字段、签名、方法体都放在 #if WX 内。
