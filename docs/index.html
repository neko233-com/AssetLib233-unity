AssetLib233

独立 Unity / 团结引擎资源热更库。YooAsset 生态无痛升级，原生支持微信/抖音/TapTap 小游戏、WebGL、HybridCLR。

Unity 2021.3+ 团结引擎 1.x HybridCLR 就绪 Runtime 零依赖
架构总览

Facade 统一入口 → AssetGroup 资源组 → Collector / Manifest / Download / Provider / Platform Plugin / GC 六大子系统

核心特性

为小游戏与 HybridCLR 场景深度优化，同时兼容 YooAsset 既有 CDN 产物

小游戏优先

微信 / 抖音 / TapTap 小游戏原生插件适配，CDN 下载并发锁 10，完整缓存链路管理，避免小游戏环境缓存表与实际文件不一致问题。

YooAsset 兼容

直接读取 YooAsset CDN 产物（Manifest / Bundle 格式），无需重新打包或迁移 CDN。从 YooAsset 迁移只需替换 API 调用层。

零依赖

Runtime 零外部依赖，可直接放入任何 Unity / 团结工程。UniTask 作为可选插件接入，不强制绑定异步方案。

真机诊断

内置 BuildRuntimeDiagnosticString()，一键输出平台、缓存、版本、内存、下载状态完整诊断串，真机排查无需附加调试器。

快速接入

三行核心调用：初始化 → 初始化资源组 → 异步加载资源

C# // 1. 初始化 AssetLib233（通常在启动场景或热更入口调用一次） await AssetLib233.Initialize(new AssetLibConfig { CdnBaseUrl = "https://cdn.example.com/res", Platform = EnumAssetPlatform.Auto, }); // 2. 初始化资源组（拉取 Manifest、校验版本、按需下载） await AssetLib233.InitializeGroup("default", new GroupInitParams { IsRequired = true, OnProgress = (loaded, total) => { Debug.Log($"下载进度: {loaded}/{total}"); } }); // 3. 异步加载资源 Sprite icon = await AssetLib233.LoadAssetAsync<Sprite>("default", "image_icon/hero_001"); myImage.sprite = icon; // 释放单个资源 AssetLib233.ReleaseAsset("default", "image_icon/hero_001");
平台支持

覆盖主流 Unity / 团结引擎目标平台，小游戏平台提供原生插件级适配

平台 宏定义 状态 说明
PC / iOS / Android - Stable 标准 UnityWebRequest / FileStream 链路，零配置开箱
WebGL UNITY_WEBGL Stable 异步加载 + IndexedDB 缓存，兼容 WebGL 2.0
微信小游戏 WX Stable 微信插件预下载缓存链路，并发 10，支持 wechatminigame-preload
抖音小游戏 DOUYINMINIGAME Stable 抖音小游戏 SDK 原生下载接口适配
TapTap 小游戏 TAPMINIGAME Stable TapTap 小游戏平台下载与缓存适配
PS5 UNITY_PS5 Planned 主机平台预留扩展点，按 NDA 要求离线部署
异步模型

三种异步方案按需选用，不强制绑定任何一种

Callback

传统回调模式，兼容 Unity 2021.3 无 async 场景

AssetLib233.LoadAssetAsync<Sprite>( "default", "icon", sprite => { img.sprite = sprite; } );

C# await

标准 async/await，适用于 .NET 4.x / IL2CPP

Sprite icon = await AssetLib233 .LoadAssetAsync<Sprite>( "default", "icon"); img.sprite = icon;

UniTask

可选插件接入，零 GC 高性能异步，推荐小游戏使用

Sprite icon = await AssetLib233 .LoadAssetUniTask<Sprite>( "default", "icon"); img.sprite = icon;
