AssetLib233-unity

独立 Unity / 团结引擎资源系统。目标是完全替代 YooAsset，优先服务微信小游戏 / WebGL / HybridCLR / 多 AssetGroup 热更场景。

快速开始 架构 热更 收集器 平台 异步 编辑器模拟 调试

快速开始

用户侧只访问单例 AssetLib233.Instance。

安装

UPM Git 地址：https://github.com/neko233-com/AssetLib233-unity.git

无 Git 导入：运行 Tools/install-assetlib233.ps1，脚本会下载 GitHub zip 并复制到 Assets/neko233/AssetLib233。

核心架构

模块职责
AssetGroup顶层资源组，承载 login / default / story 等独立热更域。
AssetCollector配置资源根目录、地址规则、标签、打包规则。
AssetManifest二进制清单，记录 asset 到 bundle、bundle 依赖、hash、crc、size、tags。

热更流程

请求版本加载 Manifest生成下载计划单文件 / 并发下载统一 Loading校验缓存清理废弃 AB

打包上传刷新一条龙

AssetLib233EditorPublishPipeline 支持 build、SSH/工具上传 CDN、CDN 刷新、报告生成、溯源服务器 HTTP 上报。私密配置来自 AssetLib233.publish.local.json 或环境变量 ASSETLIB233_LOCAL_CONFIG，不会进入 GitHub。

login 快启与登录后资源组

AssetLib233StartupPlan 原生支持一个 LoginFast 首组和 N 个登录后资源组。业务 Loading 可通过 IAssetLib233LoadingSink 以后接入；不接时流水线静默运行。

收集器规则

Collector 支持 PackTogether、PackSeparately、PackByTopDirectory、PackBySecondDirectory、PackByFileExtension、RawFile、ArchiveFile。

扩展点

压缩策略、打包规则、打包验证、下载传输层、CDN 工具、溯源上报均接口化。框架本体 0 依赖；UniTask 为可选插件。

平台插件

插件平台能力
Plugin_MiniGame_WX微信小游戏只使用 WX 宏识别平台，下载并发 10、微信缓存根路径、小游戏加载方式。
Plugin_MiniGame_DouYin抖音小游戏下载并发 10、抖音缓存根路径、小游戏加载方式。
Plugin_MiniGame_TapTapTapTap 小游戏Tap.env 缓存根路径、下载并发 10。

异步模型

AssetLib233 原生支持 callback 和 C# async/await。UniTask 通过 Plugin_UniTask 接入。

编辑器真机下载模拟

EnumAssetLib233PlayMode.EditorRemoteSimulation 用于 Editor 内模拟真机远端下载，不绑定业务 Loading。

调试工具

调试层输出 AssetLib233DebugSnapshot，用于编辑器窗口、运行时控制台、远端诊断。

Asset GC

AssetLib233 内置自动和手动 Asset GC。自动 GC 通过 AssetLib233.Instance.Tick() 推进；手动 GC 使用 AssetLib233.Instance.CollectAssets(true)。
