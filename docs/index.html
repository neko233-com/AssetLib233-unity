AssetLib233-unity

独立 Unity / 团结引擎资源系统。AssetGroup 管热更资源组，Collector 管 AB 切分，Manifest 管版本和依赖，Plugin 管平台差异。

快速开始 架构 热更 收集器 平台 异步 编辑器模拟 调试

快速开始

用户侧只访问单例 AssetLib233.Instance。

核心架构

模块职责
AssetGroup顶层资源组，承载 login / default / story 等独立热更域。
AssetCollector配置资源根目录、地址规则、标签、打包规则。
AssetManifest二进制清单，记录 asset 到 bundle、bundle 依赖、hash、crc、size、tags。

热更流程

请求版本加载 Manifest生成下载计划下载 Bundle校验缓存

收集器规则

Collector 支持 PackTogether、PackSeparately、PackByTopDirectory、PackBySecondDirectory、PackByFileExtension、RawFile、ArchiveFile。

平台插件

插件平台能力
Plugin_MiniGame_WX微信小游戏只使用 WX 宏识别平台，下载并发 10、微信缓存根路径、小游戏加载方式。
Plugin_MiniGame_DouYin抖音小游戏下载并发 10、抖音缓存根路径、小游戏加载方式。
Plugin_MiniGame_TapTapTapTap 小游戏Tap.env 缓存根路径、下载并发 10。

异步模型

AssetLib233 原生支持 callback 和 C# async/await。UniTask 通过 Plugin_UniTask 接入。

编辑器真机下载模拟

EnumAssetLib233PlayMode.EditorRemoteSimulation 用于 Editor 内模拟真机远端下载，不绑定业务 Loading。

调试工具

调试层输出 AssetLib233DebugSnapshot，用于编辑器窗口、运行时控制台、远端诊断。
