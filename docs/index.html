AssetLib233-unity

独立 Unity / 团结引擎资源热更库。目标：替代 YooAsset，原生服务微信小游戏、WebGL、HybridCLR、多 AssetGroup 热更。

首页 快速上手 对比其他框架 架构设计 详细文档 QA问题 联系我们

一句话

AssetLib233.Instance 是用户唯一入口。AssetGroup 对标 YooAsset Package，但更贴合 login 首组、登录后 N 个内容包、统一 Loading、微信小游戏真机排障、构建报告和 Asset GC。

微信小游戏优先默认并发 10Runtime 0 依赖UniTask 可选纯 HTML 文档

能力入口

能力入口说明
运行时加载AssetLib233.Instance.LoadAssetAsync<T>支持 callback、C# await、可选 UniTask。
多组下载CreateMultiGroupDownloadOperation多个 AssetGroup 汇总成一个进度，适合登录后 default + story。
真机诊断BuildRuntimeDiagnosticString输出平台、Manifest、缓存路径、下载和加载事件，便于复制日志。
打包产物AssetLib233EditorBuildPipeline生成 AB、Manifest、version、构建报告，并自动弹出报告窗口。

安装

UPM Git 地址：

https://github.com/neko233-com/AssetLib233-unity.git

无 Git 一行导入：

powershell -ExecutionPolicy Bypass -Command "iwr https://raw.githubusercontent.com/neko233-com/AssetLib233-unity/main/Tools/install-assetlib233.ps1 -OutFile $env:TEMP/install-assetlib233.ps1; & $env:TEMP/install-assetlib233.ps1 -ProjectRoot (Get-Location).Path"

核心链路

AssetGroup 包含 N 个 AssetCollector，每个 Collector 产出 N 个 AB / RawBundle / ArchiveBundle。每个 AssetGroup 产出独立 .version 和二进制 .manifest，运行时按 Manifest 下载、校验、加载、清理废弃包。
