快速上手

从安装、初始化、下载到加载，尽量只写少量业务代码。

首页快速上手对比其他框架架构设计详细文档QA问题联系我们

1. 初始化

AssetLib233PackageConfig config = new AssetLib233PackageConfig();
config.PackageName = "default";
config.PlayMode = EnumHotUpdateType.Host;
config.DefaultHostServer = "https://cdn.example.com/default";
config.FallbackHostServer = "https://backup.example.com/default";
config.EnableBundleCrypto = true;
config.BundleCryptoPassword = "root";

AssetLib233.Instance.Initialize();
AssetLib233.Instance.InitializeGroup(config);

2. 热更准备与下载

AssetLib233PreparePackageOperation prepareOperation =
    AssetLib233.Instance.CreatePreparePackageOperation("default");
await prepareOperation;

AssetLib233PackageDownloadOperation downloadOperation =
    AssetLib233.Instance.CreatePackageDownloadOperation("default", string.Empty);
await downloadOperation;

3. 加载资源

AssetHandle233<GameObject> handle =
    AssetLib233.Instance.LoadAssetAsync<GameObject>("default", "main");
await handle;
GameObject prefab = handle.AssetObject;

4. login 首组 + 登录后 N 组

AssetLib233StartupPlan plan = new AssetLib233StartupPlan();
plan.LoginGroupName = "login";
plan.PostLoginRequiredGroups.Add("default");
plan.PostLoginRequiredGroups.Add("story");
plan.PostLoginOptionalGroups.Add("voice");

AssetLib233LoginFastOperation loginOperation =
    AssetLib233StartupFacade.CreateLoginFastOperation(plan);
await loginOperation;

neko233/AssetLib233/构建选中 BuildProfile
