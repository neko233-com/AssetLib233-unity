CDN provider

内置 VolcengineChina、Aliyun、TencentCloud、Aws、Custom。默认火山引擎中国。所有 provider 默认复用 .local 中的上传/刷新命令，用户可注册自定义 IAssetLib233CdnProvider。

{
  "cdnProvider": "VolcengineChina",
  "cdnRegion": "cn",
  "cdnBucket": "your-volcengine-tos-bucket",
  "cdnPathPrefix": "project/env/weixin/minigame/StreamingAssets",
  "cdnGoToolConfigDirectory": "_tools/cdn_douyin_tools",
  "cdnGoToolConfigName": "config-for-douyin-refresh-cdn.minigame_wx_test.json",
  "cdnGoToolExecutableName": "douyin-refresh-cdn.exe",
  "cdnGoToolCommand": "run",
  "uploadToolPath": "D:/Tools/upload.exe",
  "refreshToolPath": "D:/Tools/refresh.cmd"
}

免打包验证

AssetLib233EditorAgentValidationPipeline 不执行 AB 打包，只验证 BuildProfile、Collector、资源地址、Editor 可加载性，以及 Go 工具 publish_verify 指向的已发布 CDN 文件。

powershell -ExecutionPolicy Bypass -File Assets/neko233/AssetLib233/Tools/agent-validate.ps1 -UnityPath "D:/Unity/Editor/Unity.exe" -ProjectRoot "D:/Project"

中英文输出

配置 language 为 zh-CN 或 en-US。发布报告和验证报告会输出完整上下文，包含平台、环境、配置路径、命令、日志文件、退出码、错误摘要。

Agent-first 发布

powershell -ExecutionPolicy Bypass -File Assets/neko233/AssetLib233/Tools/agent-publish.ps1 -UnityPath "D:/Unity/Editor/Unity.exe" -ProjectRoot "D:/Project"

产物报告保存到 reportRoot，报告包含 provider、bucket、路径前缀、每一步命令、日志路径、退出码、溯源结果。

清理废弃 Bundle

AssetLib233ObsoleteBundleCleaner 根据最新 Manifest 和本地 CacheIndex 生成清理计划。可先 dry-run 输出报告，再执行删除，避免误删当前版本仍在引用的 AB。

真机诊断

string report = AssetLib233.Instance.BuildRuntimeDiagnosticString();
Debug.Log(report);

诊断串会输出平台、Package、Manifest 版本、缓存根路径、最近下载事件、Bundle 加载事件、错误信息。
