微信小游戏需要哪些宏？

只需要 WX。微信 SDK 类型引用全部包在 #if WX 内，编辑器可继续模拟非 WX 环境。

Editor 正常，真机资源为空怎么办？

先打印 BuildRuntimeDiagnosticString。重点看 Manifest 版本、Bundle 是否下载成功、缓存路径、hash/size、Bundle 依赖是否加载、地址是否命中。

多个 AssetGroup 怎么显示一个 Loading？

使用多组下载 operation，把 default、story、voice 等多个 group 汇总到一个 IAssetLib233LoadingSink。

可以不用 UniTask 吗？

可以。Runtime 原生 callback 和 C# await；UniTask 在 Plugin_UniTask 中可选安装。

CDN 私密路径会进 GitHub 吗？

不会。所有 SSH、bucket、token、服务器路径放 AssetLib233.publish.local.json 或 ASSETLIB233_LOCAL_CONFIG 指向的本地文件。
